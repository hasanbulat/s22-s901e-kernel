s22-s901e-kernel
================

Build kernel
------------
` cd android_kernel_samsung_sm8450/ `

` ./build_menu`
- press `ENTER` to download necessary build tools
- select [1] Build kernel
- select [1] waipio-gki
- press `ENTER` to start the build process

Create SAMSUNG tar
------------------
- run `s901e.sh`
- tar file is locate at aik-3.8 dir
